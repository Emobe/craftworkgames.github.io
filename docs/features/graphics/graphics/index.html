Graphics

The MonoGame.Extended.Graphics library contains extensions useful for generating dynamic geometry and batching draw calls.

Installation#

MonoGame.Extended.Graphics is distributed via a NuGet package. You can add the NuGet package to your C# project through your IDE of choice (Visual Studio, Xamarin Studio, Rider, etc) or through the Command Line Interface (CLI) using the dotnet command.

dotnet add package MonoGame.Extended.Graphics
